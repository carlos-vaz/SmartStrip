## Arduino Nano Bluetooth test with the AT-09
### Pin Assignments: 
| Arduino 	| AT-09 	|
| ------------- |:-------------:|
| D2		| TX		| 
| D3		| RX		| 
| GND		| GND		| 
| 5V		| VCC		| 
| 		| Not Connected |
| 		| Not Connected |

This test expects commands from a master BLE device (e.g. Android app)
